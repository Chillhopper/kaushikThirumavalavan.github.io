Colour Tracking Bot

Overview

The Colour Tracking Bot is a cutting-edge robotics project designed to autonomously detect and follow a specific color using advanced computer vision techniques and real-time image processing.

Project Goals

  • Develop an autonomous bot capable of real-time color detection and tracking.
  • Implement computer vision algorithms to enhance precision and efficiency.
  • Integrate sensor data for improved navigation and obstacle avoidance.
  • Explore practical applications in various fields, including AGVs, search and rescue, and interactive robotics.

Technical Details

The bot uses a camera to capture images and process them to identify the target color. The image processing algorithm isolates the color and calculates the centroid, which is then used to direct the bot's movement. The project includes:

  • Real-time color detection and tracking using OpenCV.
  • Autonomous navigation based on the detected color.
  • Integration with sensors for obstacle detection and avoidance.
